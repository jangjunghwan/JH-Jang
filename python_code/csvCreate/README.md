# csvBuild.py #

## PathFunction() ##
 - 현재 경로 경로 파일 가져오기.

## pcmFilePath() ##
 - 해당 경로에 'pcm'파일 목록 하나에 List에 담기.

## textFilePath() ##
 - 해당 경로에 'txt', 'm_txt'파일 목록 각각 하나에 List에 담기.

## csvFileCreat() ##
 - csvFile Header 적용 및 생성.

## FileWrite() ##
 - csvFile에 내용 넣기.

## GenderCheck() ##
 - 입력 문제에 특정 문자열 유,무 판별.

## FileNameCheck() ##
 - 입렵 문제 Split으로 리스트로 변환.
